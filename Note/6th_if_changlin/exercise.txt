新建学生成绩单：程序可以创建一个新的学生成绩单，并将其保存在文件中。
添加学生成绩：程序可以要求用户输入学生的姓名和成绩，并将其添加到学生成绩单中。
查找学生成绩：程序可以要求用户输入学生的姓名，并在学生成绩单中查找该学生的成绩。
统计成绩：程序可以计算学生成绩单中所有学生的平均成绩。
退出程序：程序可以让用户选择退出程序。
要求：

使用 switch 语句实现菜单功能，让用户选择要执行的操作。
使用 if 语句来处理用户输入和条件判断。
使用 for 循环来遍历学生成绩单中的学生信息。
使用 while 循环来实现程序的主循环，直到用户选择退出程序。
使用逻辑运算符 && 和 || 来处理多个条件的判断。
提示：

可以使用结构体或类来表示学生的姓名和成绩。
可以使用文件来保存学生成绩单，使用文件输入/输出流来读取和写入数据。
